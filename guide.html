🔧 職員向け設定ガイド

このページでは、はたらくんに新しい業種やシナリオを追加する方法をご案内します。

① 業種を追加する

業種は index.html に直接記述されています。以下の手順でボタンを追加してください。

📍 手順

  • index.html を VS Code などで開く
  • <div class="button-grid"> の中に以下のようなボタンを追加
<button onclick="startScene('newJobId')">🧪 新しい業種名</button>

▲ VS Codeで業種ボタンを追加する位置

② シナリオを追加する

シナリオは scenarios.json に業種ごとにまとめて記述されています。以下の手順で追加してください。

📍 手順

  • scenarios.json を VS Code などで開く
  • 該当する業種（例："lightWork"）の配列の末尾に、以下の形式で新しいシナリオを追加
{
  "question": "スタッフ：「〇〇はどこに置きましたか」",
  "choices": [
    "「〇〇は△△に置きました」",
    "「思い出すので少し待ってください」",
    "自分で探してください"
  ],
  "feedbacks": [
    "◎ 丁寧な対応で安心感があります！",
    "〇 すぐに答えられない場合に思い出そうとする姿勢は良いです。",
    "× 不親切な対応です。"
  ]
}

▲ VS Codeで業種ごとの配列にシナリオを追加する位置

③ 新しい業種に対応するシナリオを追加する

新しい業種を index.html に追加した場合は、対応するシナリオも scenarios.json に記述する必要があります。

📍 手順

  • scenarios.json を VS Code などで開く
  • ファイルの最下部に、新しい業種名（例："flowerShop"）をキーとして追加
  • その業種に対応するシナリオを配列で記述
"flowerShop": [
  {
    "question": "お客様：「この花、長持ちしますか？」",
    "choices": [
      "「はい、水替えをすれば1週間ほど持ちます」",
      "「わかりません」",
      "無視する"
    ],
    "feedbacks": [
      "◎ 丁寧な説明で安心感があります！",
      "△ 他のスタッフに確認する姿勢があるとより良いです。",
      "× 無視は不親切です。"
    ]
  }
]

📌 注意点

  • 業種名（キー）は index.html の startScene('〇〇') のIDと一致させてください
  • JSON形式に注意（カンマの付け忘れ、全角記号など）
  • 業種やシナリオを追加・変更する際は、必ず変更前のデータファイルを別フォルダに保存しておきましょう。
    バックアップを残しておくことで、万が一エラーが出たり動作しなくなった場合でも、すぐに元の状態へ戻すことができます。

🎉保存後、アプリを再起動すると新しい業種とシナリオが反映されます。
